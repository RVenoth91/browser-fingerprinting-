# 🆔 Browser Fingerprinting - Détecteur d'Empreinte Digitale

Un projet de cybersécurité qui identifie les utilisateurs par les caractéristiques uniques de leur navigateur et système.

## 🎯 Comment ça marche ?

Le programme collecte :
- Système d'exploitation
- Résolution d'écran
- Polices installées
- Fuseau horaire
- Langue du système

Ces informations créent une **empreinte unique** (hash SHA256) qui identifie l'utilisateur.

## 🚀 Utilisation

```bash
python fingerprint.py
